# IB Fresh Fish App Icon Instructions

## Current Status:
- App Name: IB Fresh Fish
- Package: com.ibfreshfish.mobile
- Theme Color: #0EA5E9 (Sky Blue)

## Icon Requirements:
1. **icon.png** - 1024x1024px (main app icon)
2. **adaptive-icon.png** - 1024x1024px (Android adaptive icon)
3. **splash-icon.png** - 1024x1024px (splash screen)

## Quick Icon Options:

### Option A: Online Generator (Recommended)
1. Visit: https://icon.kitchen/
2. Choose "Emoji" or "Clipart"
3. Search for fish (üêü) or use fish clipart
4. Set background to: #0EA5E9
5. Download all sizes
6. Replace files in mobile/assets/

### Option B: Simple Design
- Blue circular background (#0EA5E9)
- White fish symbol in center
- Clean and professional

### Files to Replace:
```
mobile/assets/icon.png
mobile/assets/adaptive-icon.png
mobile/assets/splash-icon.png
```

## After Updating Icons:
Run: `npx expo prebuild --clean` to regenerate native files
